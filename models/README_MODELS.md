# Modelos entrenados

Esta carpeta está destinada a almacenar los modelos entrenados del proyecto.

Modelos previstos:
- `modelo_logistico_prestaciones.pkl`: modelo principal de clasificación (Regresión Logística).
- `arbol_prestaciones.pkl`: modelo de apoyo interpretativo (Árbol de Decisión).

Los modelos se generan al ejecutar la notebook:
`notebooks/EXAMEN_PARCIAL_AA_CASSANO_NANCY_JULIETA_ENTREGA3_MODELOS.ipynb`
